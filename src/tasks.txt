Є лінки :
/users 
/posts 
/albums 

Вложеність :
App > Users > User + UserDetails
App > Posts > Post + PostDetails
App > Albums > Album + AlbumDetails

Отримувати дані з jsonplaceholder:

1) Для users за допомогою запиту відобразити всіх користувачів. 
Відобразити деталі одного юзера при натисканні на лінк біля нього. ( Використовуючи основний запит - передавати  state  користувача в Link). 


2) Для posts за допомогою запиту відобразити всі пости. 
Відобразити деталі одного поста при натисканні на лінк біля нього. ( Використовуючи під запит  та  без використання  state   в Link). 

3) Для albums за допомогою запиту відобразити всі альбоми. 
Відобразити деталі одного альбому при натисканні на кнопку біля нього. ( Використовуючи під запит  та без використання  state   в Link).
